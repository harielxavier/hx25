==============================================
EMERGENCY TROUBLESHOOTING
==============================================

IF RANKINGS DROP:
1. Check Google Search Console for errors
2. Verify sitemap is still submitted
3. Check if website is down
4. Review recent changes to site
5. Look for Google algorithm updates
6. Check competitors (did they improve?)

IF TRAFFIC DROPS:
1. Seasonal variation (weddings are seasonal!)
2. Check Google Analytics for technical issues
3. Review top landing pages
4. Check mobile usability
5. Verify tracking code is working

IF REVIEWS STOP COMING:
1. Increase request frequency
2. Make links more prominent
3. Offer incentive (free print)
4. Try different request method (text vs email)
5. Ask in person at meetings

IF CONVERSIONS DROP:
1. Check contact form is working
2. Test phone number clicks
3. Review pricing page (too expensive?)
4. Check competitor pricing
5. Improve call-to-actions
